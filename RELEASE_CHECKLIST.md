# FishScript public release checklist

## Before release

- [ ] Replace `YOUR_USERNAME` in `website/index.html`
- [ ] Test IDE with `fishscript ide`
- [ ] Test CLI with `fishscript run fishscript/examples/hello.fish`
- [ ] Build Windows app with `build_windows.bat`
- [ ] Optional: build installer with Inno Setup using `installer/FishScript.iss`
- [ ] Create GitHub repository
- [ ] Push code
- [ ] Enable GitHub Pages
- [ ] Create GitHub Release
- [ ] Upload `FishScriptInstaller.exe`

## Suggested repository description

FishScript: the world’s most unnecessary aquarium programming language.

## Suggested website tagline

Code inside an aquarium.
